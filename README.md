## Installation

```sh
git clone https://github.com/Terranomicon/REST-API-Address-Book.git
```

```
composer update
```

```
sudo php -S localhost:80 index.php
```

```
structure in dump.sql
```
Add config.ini as default.config.ini
```
configure config.ini
```

Для заполнения бд скрипт
```
php seed.php
```
